<h1>Сайт онлайн-магазина принтов</h1>

Сайт-визитка, созданный на основе макета в Figma с применением адаптивной вёрстки.

Цель проекта — самообучение, а также проверка и закрепление практических навыков. В рамках данного проекта была реализована адаптивная вёрстка с целью углубления понимания её принципов работы.
